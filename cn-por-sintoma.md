## API utilizada

- **Base URL**: `https://cima.aemps.es/cima/rest`
- **Endpoint 1 (búsqueda)**: `GET /medicamentos?nombre=<principio_activo>&receta=0&comerc=1&pagesize=25`
- **Endpoint 2 (códigos nacionales)**: `GET /presentaciones?nregistro=<nregistro>`
  - Se filtra cada presentación a `comerc==true && receta==false`.

Para cada síntoma se mapean varios principios activos (basado en `src/lib/symptoms.ts` y guías de medicación sin receta). Para cada activo se pide la búsqueda por nombre, se recogen los `nregistro` y luego sus presentaciones (CN).
